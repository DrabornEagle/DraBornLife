# DraBornLife

DraBornLife, Antalya’ya sıfırdan yeni hayat kurma sürecini takip etmek için geliştirilen kişisel mobil uygulamadır.

## Güncel kararlar

- Geliştirme yönü: Expo / React Native
- Test yöntemi: Expo Go
- İlk APK / AAB çıktısı: v1.0
- v0.1, v0.2 ve v0.3 sürecinde APK alınmayacak
- Web sürüm yok
- Supabase / cloud database yok
- Veri saklama: cihaz içinde lokal veri
- Yedekleme: manuel dışa aktar / içe aktar
- Google Drive yedek: v1.0 sonrası opsiyonel
- Android paket adı: com.draborneagle.drabornlife
