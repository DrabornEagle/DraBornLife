# DraBornLife

DraBornLife is a personal mobile app for planning the Antalya relocation and new life setup.

## Current decisions

- Expo / React Native
- Test with Expo Go
- First APK/AAB at v1.0
- Mobile app only
- Local device data
- Manual export/import backup
- Optional Google Drive backup after v1.0
- App id: com.draborneagle.drabornlife
