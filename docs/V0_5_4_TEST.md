## Form dogrulama kontrolu

- [ ] Para ekraninda bos baslik kaydedilmez
- [ ] Para ekraninda 0 veya negatif tutar kaydedilmez
- [ ] Gelirde birikime ayrilan tutar gelirden buyuk olamaz
- [ ] Liste ekraninda bos baslik kaydedilmez
- [ ] Liste ekraninda adet 1 veya daha buyuk olmali
- [ ] Liste ekraninda negatif fiyat kaydedilmez
- [ ] Borc ekraninda bos baslik kaydedilmez
- [ ] Borc ekraninda toplam borc 0'dan buyuk olmali
- [ ] Borc ekraninda hedef tarih YYYY-MM-DD formatinda olmali
- [ ] Borc odemesi kalan borctan buyuk olamaz
- [ ] Ozel hedefte baslik en az 3 karakter olmali
- [ ] Ozel hedefte biriken tutar varsa tahmini butce de olmali
- [ ] Ozel hedefte biriken tutar butceden buyuk olamaz
- [ ] Ozel hedefte tarih sayiyla basliyorsa YYYY-MM-DD formatinda olmali
- [ ] Ayarlar ekraninda aktif yil ve fiyat kontrolleri calisir
