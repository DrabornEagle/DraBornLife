# DraBornLife v1.2 Sürüm Planı ve Yapılacaklar Listesi

## v1.2 ana hedef

v1.2 sürümünün ana hedefi, v1.0 tasarım kapanışı ve v1.1 stabilizasyon planından sonra DraBornLife'ı gerçek kullanıma daha uygun hale getirmektir.

Bu sürümde odak:

- Gerçek Expo Go cihaz testinden gelen düzeltmeler
- Aile aktivite / aquapark / sahil planı
- Ev kurulum odaları ve eşya grupları
- Antalya bölge ve yaşam notları
- Birikim hedefleri için haftalık / aylık öneriler
- Borç kapatma planı
- Manuel yedek deneyimi iyileştirme
- Genel hata düzeltmeleri

## Kesin APK kararı

v1.2 planı APK anlamına gelmez.

APK build süreci yine sadece kullanıcı açıkça şu komutu verdiğinde başlar:

```text
APK almaya başlayalım
```

Bu komut gelmeden:

- APK alınmayacak.
- EAS build başlatılmayacak.
- Android versionCode artırılmayacak.
- Build ayarları açılmayacak.
