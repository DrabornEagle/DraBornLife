# DraBornLife v0.0.3 Tasarım Sistemi

## Konsept

DraBornLife artık sadece finans takip ekranı gibi değil, Antalya’ya yeni hayat kurma hissini veren premium bir yaşam paneli gibi tasarlanacak.

Tasarım hissi:

- Miami havası
- Palmiye
- Deniz
- Gün batımı
- Ferah, iç açıcı arayüz
- Premium ama sıcak mobil uygulama dili
