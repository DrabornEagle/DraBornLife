## Tasarim dili

### Genel atmosfer

- Antalya sahil yasami + Miami palm vibe.
- Fazla karanlik, karmasik veya oyun HUD gibi gorunen arayuz yok.
- Daha temiz, ferah ve premium mobil uygulama hissi.
- Kartlar arasinda yeterli bosluk.
- Yazilar okunur, nefes alan ve bolum bolum ayrilmis olmali.

### Renk yonu

Ana his:

- Deniz mavisi
- Turkuaz
- Kum / krem tonlari
- Palmiye yesili dokunuslar
- Sicak gun batimi vurgu rengi

Kurallar:

- Arka plan goz yormayacak.
- Ana aksiyon butonlari net gorunecek.
- Kritik finans/borc bilgileri okunur ve ayirt edilebilir olacak.
- Renkler cok neon veya karmasik olmayacak.

### Tipografi

- Basliklar daha net ve guclu.
- Aciklama metinleri kucuk ama okunur.
- Kart icindeki veriler hizli taranabilir.
- Finans rakamlari buyuk, hizli okunur ve hizali.
- Gereksiz uzun paragraflar ekranlarda azaltilacak.

### Kart sistemi

- Tum ana veriler kart mantigiyla sunulacak.
- Kartlar ayni gorsel dile sahip olacak.
- Yuvarlatilmis kose, hafif golge ve temiz bosluk kullanilacak.
- Her kartin tek amaci olacak: ozet, hedef, borc, liste, yedek, test gibi.

## GitHub branch ve yedek kurali

Her yeni surume gecmeden once mevcut ana branch yedek branch olarak alinacak.

Bu surum icin yedek branch:

```bash
backup-v1.0.1
```

Geri donus mantigi:

- Sorun cikarsa yeni surum temizlenir.
- Onceki yedek branch uzerinden temiz geri donus yapilir.
- Yedek branch sadece guvenli geri donus noktasi olarak tutulur.

## Termux kurali

Kalici klasor adi her zaman:

```bash
~/DraBornLife
```

Kurallar:

- Kurulum sadece GitHub zip uzerinden yapilacak.
- `DraBornLife-main` kalici klasor olarak kalmayacak.
- Python yok.
- patch yok.
- jdk yok.
- perl yok.
- /tmp yok.
- APK build yok.
- Expo Go ile devam.

## Termux tek seferlik kurulum + lokal yedek komutu

```bash
cd ~ && [ -d DraBornLife ] && cp -a DraBornLife DraBornLife-backup-v1.0.1-$(date +%Y%m%d-%H%M%S) || true && rm -rf DraBornLife DraBornLife-main DraBornLife-main.zip && curl -L https://github.com/DrabornEagle/DraBornLife/archive/refs/heads/main.zip -o DraBornLife-main.zip && unzip -q DraBornLife-main.zip && mv DraBornLife-main DraBornLife && rm -f DraBornLife-main.zip && cd DraBornLife && npm install && npx expo start
```

## Termux geri alma komutu

```bash
cd ~ && LAST_BACKUP=$(ls -dt DraBornLife-backup-v1.0.1-* 2>/dev/null | head -n 1) && [ -n "$LAST_BACKUP" ] && rm -rf DraBornLife && cp -a "$LAST_BACKUP" DraBornLife && cd DraBornLife && npx expo start || echo "v1.0.1 icin lokal yedek bulunamadi."
```
